# BizConnect – Business Networking & Referral Platform

A modern, fully responsive landing page for a Business Networking & Referral Platform. Built as part of an internship assessment.

## 📋 Project Description

BizConnect is a landing page designed for a structured business networking platform that helps professionals:
- Connect through chapter-based networking
- Attend structured meetings
- Exchange high-quality referrals
- Track business growth via a performance dashboard

## 🛠 Technologies Used

- **React 18** – Component-based UI (beginner–medium level)
- **Custom CSS only** – No frameworks like Bootstrap or Tailwind
- **Vite** – Fast development server and build tool
- **Google Fonts** – Playfair Display + DM Sans

## 🚀 Setup Instructions

### Prerequisites
- Node.js v18+ installed

### Steps

```bash
# 1. Extract the zip / clone the repo
cd biz-network

# 2. Install dependencies
npm install

# 3. Run development server
npm run dev
# Opens at http://localhost:5173

# 4. Build for production
npm run build

# 5. Preview production build
npm run preview
```

## ✅ Features Implemented

- [x] Sticky responsive Navbar with mobile hamburger menu
- [x] Hero section with stats card and trust indicators
- [x] How It Works – 4-step process with hover effects
- [x] Key Features – 6 feature cards with icons
- [x] Why Choose Us – split layout with stat cards
- [x] Who Should Join – audience grid cards
- [x] Testimonials – 3 member reviews
- [x] Final CTA section
- [x] Footer with contact info, quick links
- [x] CSS animations (fadeUp on hero load)
- [x] Fully responsive: desktop → tablet → mobile
- [x] CSS custom properties (variables) for easy theming

## 🎨 Design Decisions

- **Color palette**: Navy blue (`#1a3c5e`) + Gold accent (`#e8a020`) for a professional and trustworthy feel
- **Typography**: Playfair Display (headings) + DM Sans (body) for a refined yet readable look
- **Layout**: CSS Grid + Flexbox — no frameworks
- **Responsiveness**: Breakpoints at 1024px and 768px
